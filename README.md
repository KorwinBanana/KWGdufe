**-----茶珂-----**


连通学校各类信息系统，用于查询各类在校信息
**（感谢师兄的接口[小广财](http://www.wegdufe.com/)）**


**-----主要功能-----**

* 每天推送明日课程
* 每日反馈校园卡余额
* 每周提醒当前借阅
* 快速查课、各类学期课表
* 掌握上课地点时间
* 实时了解成绩和绩点
* 完成图书续借
* 获取个人在校信息
